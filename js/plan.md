# **🧠 PLAN DE APRENDIZAJE – 4 MESES (10 h / semana)**

⏱️ **Total:** \~160 horas  
 🎯 **Objetivo:** Que al final puedas **ofrecer servicios reales** con React / Next.js

---

## **🔹 MES 1 – JavaScript desde 0 (EL MÁS IMPORTANTE)**

⏱️ 10 h / semana

### **Qué aprender (en orden)**

* Variables (`let`, `const`)

* Funciones

* Condicionales

* Arrays y objetos

* Métodos clave:

  * `map`

  * `filter`

  * `find`

* Eventos

* DOM

* `fetch`, `async / await`

### **Curso recomendado (en español, muy claro)**

⭐ **JavaScript Moderno**   
 👉 Es el **mejor explicado**, va paso a paso y con ejemplos reales.

💡 Consejo clave:  
 No avances rápido. **Este mes define todo**.  
 Si JS no queda claro → React se vuelve frustrante.
